## Environment

This is a Python 3.14 project with a virtual environment located at `.venv/`.

## Development Setup

Activate the virtual environment:
```bash
source .venv/bin/activate
```

Install dependencies (when requirements are added):
```bash
pip install -r requirements.txt
```

This is currently a minimal Python project. The main code is in `airtable.py`.
